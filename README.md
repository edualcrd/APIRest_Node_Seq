# API Rest con AutoCRUD y Arquitectura MVC Reducida

Este proyecto implementa una API REST con Node.js y Sequelize que utiliza un generador automático de código (AutoCRUD) para crear una arquitectura MVC reducida (Rutas, Controladores Extendidos, Controladores Base y Servicios).

## Requisitos Previos
- Node.js instalado
- Base de datos MySQL corriendo
